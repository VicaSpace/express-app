# Installation

 1. Install the latest version of npm: 
	```sh
	npm install npm@latest -g
	 ```
 2. Clone source repository: 
	```sh
	git clone https://github.com/VicaSpace/express-app.git
	```
 3. Install needed dependencies: 
	```sh
	npm i
	```
 4. Start the server: 
	```sh
	npm start
	```
